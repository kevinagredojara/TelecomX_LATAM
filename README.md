# Telecom X — Análisis de Evasión de Clientes (Churn)

## Objetivo
Analizar la evasión de clientes (Churn) para identificar patrones asociados a variables categóricas y numéricas, apoyando estrategias de retención.

## Contenido
- `telecomx_churn_analysis.ipynb`: Notebook con limpieza de datos, análisis exploratorio y conclusiones.

## Flujo de trabajo
1. Carga de datos desde API (JSON)
2. Normalización y limpieza
3. Análisis exploratorio (EDA)
4. Visualización de churn
5. Insights y recomendaciones

## Principales análisis
- Distribución de evasión
- Churn por variables categóricas
- Churn por variables numéricas (tenure)

## Tecnologías
- Python
- pandas
- matplotlib
- seaborn

## Ejecución
El notebook puede ejecutarse directamente en Google Colab.
